# cloud-storage-api

## Env variables
```
GOOGLE_AUTH_CLIENT_ID=
GOOGLE_AUTH_CLIENT_SECRET=

JWT_ACCESS_SECRET=
JWT_REFRESH_SECRET=

POSTGRES_HOST=
POSTGRES_PORT=
POSTGRES_USER=
POSTGRES_PASSWORD=
POSTGRES_DATABASE=cloud-storage
PGDATA=/var/lib/postgresql/data/pgdata

MINIO_ENDPOINT=
MINIO_PORT=
MINIO_ACCESSKEY=
MINIO_SECRETKEY=
MINIO_BUCKET=
MINIO_ROOT_USER=
MINIO_ROOT_PASSWORD=
```
## Usage
Run with `docker-compose`
