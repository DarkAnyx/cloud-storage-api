# cloud-storage-api

## Env variables
```
GOOGLE_AUTH_CLIENT_ID=
GOOGLE_AUTH_CLIENT_SECRET=
JWT_ACCESS_SECRET=
JWT_REFRESH_SECRET=
POSTGRES_HOST=
POSTGRES_PORT=
POSTGRES_USER=
POSTGRES_PASSWORD=
POSTGRES_DATABASE=cloud-storage
PGDATA=/var/lib/postgresql/data/pgdata
```
## Usage
Run with `docker-compose`
